# To-do List

## Instalação

### Client

  1. `$ cd to-do-list-project`
  2. `$ npm install` ou `$ yarn install`
  3. `$ npm run dev`
  
### Server (JSON-Server)

  1. `$ cd to-do-list-project/src/services`
  2. `$ npx json-server data.json`
  
## Stacks utilizadas

### Front-end

 + `React`
 + `Context API`
 + `BrowserRouter`
 + `Java Script`

### Back-end

+ `Json-Server`
